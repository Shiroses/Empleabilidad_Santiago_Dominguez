## 🛠️ Tecnologías Utilizadas

*   **JavaScript (ES6+):** Lógica de interacciones y componentes dinámicos.

### Backend & Herramientas (Enfoque)
*   **Automatización:** Experto en **n8n** y flujos de trabajo con webhooks.
